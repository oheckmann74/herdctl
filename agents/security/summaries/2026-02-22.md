## 🚨 Critical Findings

### NEW: Finding #011 - Unauthenticated Web Dashboard APIs
**Severity**: MEDIUM-HIGH
**Priority**: P1 (before production deployment)

**Problem**: All web dashboard APIs completely unauthenticated
- 6+ REST API endpoints with no auth checks
- WebSocket server accepts all connections
- CORS limited to localhost (insufficient - any user on system can access)
- No rate limiting (DoS vulnerable)

**Exposed Data**:
- Agent configurations (Docker params, system prompts, tool access)
- Job execution history and output (may contain secrets)
- Schedule definitions and trigger capabilities
- Chat conversation history
- Fleet composition and status

**Risk**: Information disclosure + unauthorized operations + DoS

**Recommendation**: Design and implement token-based or session-based authentication framework before wider deployment.

**Full details**: [intel/2026-02-22.md](intel/2026-02-22.md#-new-finding-011-unauthenticated-web-dashboard-apis)
